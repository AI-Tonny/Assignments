Тема: MVC
Завдання 1
Створіть клас «Взуття», який має зберігати таку інформацію:
    тип взуття:
    чоловіче;
    жіноче;
    вид взуття (кросівки, чоботи, сандалі, туфлі тощо);
    колір;
    ціна;
    виробник;
    розмір.
Створіть необхідні методи для цього класу. Реалізуйте MVC-патерн для класу Взуття та код для використання моделі, контролера та представлення.

    список інгредієнтів;
    назва кухні (італійська, французька, українська тощо).
Створіть необхідні методи для цього класу. Реалізуйте MVC-патерн для класу «Рецепт» та код для використання моделі, контролера та представлення.
